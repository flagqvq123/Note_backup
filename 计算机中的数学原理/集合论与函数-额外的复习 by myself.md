# 1.集合的概念
**一般的，把确定的、可区分的事物放在一起，构成集合**
- 组成一个集合的各个元素之间是彼此不同的。
- 集合的元素是无序的。
- 一个事物是否属于一个集合，回答是确定的。（要么是要么不是）
- 集合的元素可以是任何事物，包括另外的集合。

# 2.集合间的关系
### 2.1 子集
设A、B是任意两个集合，如果A中的每一个元素都是B中的元素，则称A是B的子集合，简称子集。也称A被B包含，或B包含A。记作`A⊆B`或`B⊇A`。
用**符号表示**为：
`A⊆B⇔∀x(x∈A → x∈B)`

设A，B是两个集合，如果A⊆B且B⊆A，则称A与B相等，记作`A=B`。

==**重要原则**==：要证明两个集合相等，唯一的方法就是证明每一个集合中的任一元素均是另一个集合的元素。

设A、B是两个集合，如果A是B的子集，而B中至少有一个元素不属于A，则称A为B的真子集，记作`A⊂B`。
`A⊂B ⇔ ∀x(x∈A→x∈B) ∧ ∃x(x∈B∧x!∈A)`
`A⊂B ⇔ A⊆B ∧ A!=B`

# 3.集合的运算
	集合的运算，就是以集合为对象来得到另外一些新集合的过程。基本运算有并、交、相对补、绝对补、对称差等。
### 3.1 并交以及扩展
$A \cap B=\{x|x\in A \vee x \in B \}$
两个集合的并可以推广到n个集合的并：
设A1，A2，⋯，An是任意n个集合，则这n个集合的并可简记为:
$\bigcup_{i=1}^{n}A_i = A_1 \cup A_2 \cup ... \cup A_n$

$A\cap B = \{ x|x\in A \wedge x \in B \}$
两个集合的交可以推广到n个集合的交：
$设A_1,A_2,...,A_n是任意n个集合，则这n个集合的交可简记为\bigcup_{i=1}^{n}A_i$
并、交运算还可以推广到无穷集合情况。
