### 球盒模型：n个球放在m个盒子里
- 球：函数的定义域，只能放在一个盒子里
- 盒：函数值域，盒子里可有多个球

| 球盒      | 无限制   | 每个盒子最多一个球 | 每个盒子至少一个球 |
| ------- | ----- | --------- | --------- |
| 球盒都不区分  |       | 1 or 0    |           |
| 球不区分盒区分 |       | 1 or 0    |           |
| 球区分盒不区分 |       |           |           |
| 球盒都区分   | $m^n$ | $(m)_n$   |           |


##### 基础计数：
- 将26个字母组成5个字母的单词：
	- 每个位置都有26个选择，所以一共26^5个选择
- 将26个字母组成5个字母的单词，但是不重复
	- 第二个位置不能使用第一个位置使用的字母，所以26\*25\*24\*23\*22个选择
- 这里位置是球，字母是盒子。

- 对问题1做抽象：5，26
- 令N为n元集合，M为m元集合，且n>=0, m>=1。所有可能的映射f : N -> M数量为m^n
	- 数学归纳法证明：
		- n=0：f = 空；m^0 = 1
		- n=k+1，对n=k时有m^k种映射。此时新增的元素有m种选择，所以有m^k+1 种选择。
