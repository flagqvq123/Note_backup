# 或者“宽”或者“高”
	回顾：偏序集(S，<=)
		1.当集合有限时，一定有最大元最小元
		2.一定能扩充为线性序

### 定义：链与反链
	链(chain)：如果对任意𝑥,𝑦 ∈ 𝐴, 𝑥 ≼ 𝑦 或  𝑦 ≼ 𝑥。
	反链(antichain)： 如果对任意 𝑥 ≠ 𝑦 ∈ 𝐴, 𝑥 ⋠ 𝑦。
	它们描述偏序集中两类子关系：可以全部比较或不可比较

1) 极大元或极小元的集合组成反链
		极大元本身就是没有比它更大的元素，不可能两两比较

### 定义：最大独立集和最长链
	给定有限偏序集 P = (S,<=)
	𝛼(𝑃) = max{ |𝐴| : 𝐴是𝑃上的反链（独立集）}
	𝜔(𝑃) = max{ |𝐴| : 𝐴是𝑃上的链}
	因为有限，且一定存在最小值1，所以一定存在。
	(潜在地来说，这里运用了自然数是一个线性序，所以在有限的情况下一定存在最大值)

### ==Mirsky's theorem定理==：最长链长度 = 最小反链划分数
	给定有限偏序集 𝑃 = (𝑆, ≼) ，将𝑆划分成若干不相交的反链集，取最小划分数𝑡, 则t = 𝜔(P)

1) 划分：将集合划分为几个子集，划分的块并起来为整个集合且互相之间不相交。

##### ==证明==：
要证明 t = 𝜔(P), 即证 t >= 𝜔(P) 且 t <= 𝜔(P)

1) t >= 𝜔(P):
	简单的容斥原理：链里面每个元素都相互可比较，所以每一个元素都被包含在一个独立的反链中。且仍有可能存在其他反链。
2) t <= 𝜔(P):
	1. 仅需找到一条链比已知的最小反链划分数大于或等于的链即可。
	2. 给出一个反链的划分方式：每次选择`Ai = S的极小元集合`
		在有限次划分后，可以将整个集合划分成m个反链。
	3. 进行m次划分一定存在一条通路，从极小元到极大元一共长度为m，构造出一个长度为m的链。
	4. 这种反链构造方式的划分数 m >= t, 最长的链 𝜔(P) >= m
	5. 所以 𝜔(P) >= t

(具体的构造方式参考20-22页)

>[!tips] 一些自己的思考：
>其实这本质是一个==构造性==的证明：从极小元开始向上划分直到划分结束，此时一定有一个最长链：
>1. 为什么这个链是最长链？ 因为如果存在比它更长的，那么说明还没划分完。
>2. 为什么这种划分方式是最小划分？ 因为最长链上所有元素都互相可比较，所以最小的划分方式就是每一个独立的划分都包含上面的一个点。

##### ==推论==：给定有限偏序集 P = (S, <=), 𝛼(𝑃) \* 𝜔(𝑃)  >= |S|
	对任意有限偏序集 𝑃 = (𝑆, ≼)，𝛼(𝑃)或𝜔(𝑃)之一至少为|𝑆|^1/2。 
	直观：任意有限偏序集或者“宽”，或者“高”。 

### 定理的应用

>1. 本质是定义一个新的偏序关系：既在顺序上靠后又在大小上更大的定义<=。
>2. 此时这一整个序列可以看成是一个偏序集。
>3. 如果存在𝜔(P)(最长链长度) > |S| ，那么此时就能找到一个非递减的单调子序列。
>4. 如果存在𝛼(P) > |S| , 则存在一系列独立集。

`在讨论上面几个问题的时候，我们发现链与反链之间似乎有一种对称性。那么我们将 Mirsky's theorem 定理中的链与反链调换，这个定理是不是还成立呢？`
### Dilworth定理:
	给定有限偏序集 𝑃 = (𝑆,≼)，max{|A|: A是𝑃上的反链} = min{|ß|: ß是𝑆的链划分}
