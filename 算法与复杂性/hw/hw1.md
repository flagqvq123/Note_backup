1. 如果开始时棋子数量是4n个，则A必定会拿到最后一个棋子。如果开始时不是4n个，则A应在每次拿棋子后让棋子数目保持4n个。
	证明：
	归纳假设：开始棋子数为4n时A输，为4n-1、4n-2、4n-3时A胜。
	1. 开始时棋子数为1，2，3时，A可以一次拿完，此时A胜。
	2. 开始时棋子数为4时，A不论怎么拿，B都能拿到最后一个棋子，A输。
	3. 开始时棋子数为4n+4时，A无论怎么拿，B都可以让棋子保持到4n，由归纳假设A输。
	4. 开始时棋子数为4n+3、4n+2、4n+1时，A都可以选择拿3、2、1颗棋子使棋子数到4n，B不论怎么拿棋子都会使棋子数在4n-1、4n-2、4n-3几种情况，由归纳假设A胜。

2. 不论哪种方法，掰开所有小块的步数等于m\*n-1
	证明：
	归纳假设：先给出一个表格方便理解，用(m,n)表示它的大小为m行，n列

| m\n | 1   | 2   | 3   | 4   | 5   |
| --- | --- | --- | --- | --- | --- |
| 1   | 0   | 1   | 2   | 3   | 4   |
| 2   | 1   |     |     |     |     |
| 3   | 2   |     |     |     |     |
| 4   | 3   |     |     |     |     |
| 5   | 4   |     |     |     |     |
	1. 先证明第一行满足。
		1. 巧克力为(1,1)时，步数为1。
		2. 当前n项满足时，巧克力为(1,n+1)时，此时第一步掰开的任意一种组合(1,x), (1,n+1-x) ，它们都满足将它掰开的步数一定是x-1, (n+1-x)-1。所以将1\*(n+1)掰开需要1+x-1+(n+1-x)-1 = n步。
	2. 同理，第一列也满足。
	3. 再证明第m+1行满足
		1. 巧克力为(m+1,1)时，由(2)得满足
		2. 当m行的前n列满足时，巧克力为(m+1,n+1)时，此时第一步掰开的任意组合(m+1-x, n+1), (x, n+1) 或(m+1, n+1-y), (m+1, y)都在已经证明的满足域中。它们掰开的步数和为(m+1)\*(n+1)-2，加上第一步分开所需的一步，正好满足条件。
	4. 得证。

3. 归纳证明：假设前10\*2^n正整数都满足此条件。
	1. n=0时：6=1+5，7=2+5，8=3+5，9=4+5
	2. n+1情况：由于前10\*2^n正整数都满足条件，所以仅需考虑10\*2^n+1到10\*2^(n+1)之间的数就可以了
		1. 10\*2^(n+1)本身在数列中，满足。
		2. 其他数可以表示成 10\*2^n+m 形式，其中m满足 10\*2^(n-1) < m < 10\*2^n。由于它比10\*2^n小，且在n=n的情况中，所以一定能被除开10\*2^n的数列中的不同数加和表示。所以10\*2^n+m也可以被数列中的不同数加和表示。
	3. 得证

4. 从简单的情况开始考虑：
	1. 考虑仅有1、2号的情况：2号已经占据半数，他一定会把所有金币占为己有。
	2. 加入3号：由于1号在2号后面，如果3号被喂鲨鱼那么他就只能得到0金币。所以三号只要给1号1金币就能得到1号的票，同时将剩下的金币占为己有。
	3. 加入4号：由于3号在4号后面，如果4号被喂鲨鱼那么2号就因为情况(2)一块金币都得不到。所以四号只需要给2号1金币就能得到2号的票，同时将剩下的金币占为己有。
	4. 加入5号：由于4号在5号后面，如果5号被喂鲨鱼那么1、3号就会因为情况(3)一块金币都得不到。所以5号只需要给1、3号一块金币就能得到他们的票，同时将剩下的金币占为己有。
	5. ...
	6. 加入10号：仅需要给2、4、6、8号一块金币就能得到他们的选票，所以自己能分到96金币
	7. 如果海盗数为20，50，100，200时，第一个海盗可以得到91、76、51、1金币。如果海盗数为400，不论怎么分都有一半以上的海盗分不到金币，一定会被处死。
