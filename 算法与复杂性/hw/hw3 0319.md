1. 存储三个序列积：当前最大负序列，当前最大正序列，最大历史序列从第一个元素开始向前遍历：
	1.若下一个值为0：当前最大正、负序列设为0，更新最大历史序列
	2.若下一个值为正数：最大正负序列分别乘以这个值，更新最大历史序列
	3.若下一个值为负数：最大正负序列分别乘以这个值，互相反转为对方的值，更新历史最大序列。

2. 建立变量当前最小极小值、当前最大极大值，最大收益值
	1. 若发现了一个极小值min\[m]，比较已存储的极小值，如果大于它则舍去。如果小于它则立即将当前最小极小值和当前最大极大值配对，与最大收益值比较并保留最大值，并将当前最大极大值归0。
	2. 若发现了一个极大值max\[m]，比较已存储的极大值，如果小于它则舍去，大于它就替换之。
	3. 初始点与结束点：初始点试验下一个点的大小，若下一个点比它大则看作极小值点，若比它小则无操作。结束点试验前一个点的大小，若前一个点比它大则无操作，若比它小则看作极大值点。
	4. 遍历结束后，将当前最小极小值和最大极大值配对与最大收益值比较，保留最大项。

3. 这是一个动态规划问题。
	1. 当n=1时，仅需一步。记此时的操作为X(1,A,C,B)，表示将A上的最顶上一个圆盘直接移动到C。后面使用的“+”仅表示顺序的连接。 当n=n+1时，需要将前n个圆盘从A移动到B，将第n+1个圆盘移动到C，再将前n个圆盘从A移动到C。即X(n+1,A,C,B) = X(n,A,B,C) + X(1,A,C,B) + X(n,B,C,A)。递归即可得到结果。由于递归过程存在复用，可以构造一维数组储存X(n,a,b,c)（参数a,b,c仅有代入关系，所以不需要额外构造维度），在首次调用时进行计算并储存。
	2. 注意到在1问中实际上得到了一个递归树。仅需要在进行递归遍历的过程中进行匹配并统计步数，如果匹配成功则输出步数，匹配失败则输出-1。

4. 同样构造一个二维数组进行存放。每一个节点P(n,K)可以来自于P(n-1,K)或者P(n,K-kn)（k为所有可能正整数）。自顶向下遍历即可。

5. P(n,K)此时表示一个具体的价值，表示在考虑装n个物品后背包重量恰为K时的最大价值。
	1. P(n,K) = max(P(n-1,K), P(n-1,K-Kn) + value\[n])
	2. 初始值：P(0,0) = 0, P(0,K0) = value\[0]
	3. 根据这个式子遍历求解即可。

6. 注意到最大值就是最大绝对值的平方，所以只需要记录前n个数中最大绝对值的数A\[i]即可，B\[n] = A\[i]^2

7. 二叉树搜索0，找到的位置左右两个元素比较即可。

	1. 从左开始遍历：设最左边的数是x，向前跳跃|z-x|个数。直到找到z为止，或者到最右边为止。裁剪掉刚刚跳跃的部分（其中不可能有z）。再从右边开始进行相似的操作。
	2. 此时得到了一个两端为z的片段。在中间随机取一个点，如果为z则保留并分割为两个片段，再对每一个片段执行2。如果不为z，向左右分别执行1，直到找到z，形成两个片段。再对每个片段执行2。如果片段长度为1或者2则不再进行操作。
	3. 时间复杂度：最坏Θ(n)，最好Θ(1)，平均不会算。
