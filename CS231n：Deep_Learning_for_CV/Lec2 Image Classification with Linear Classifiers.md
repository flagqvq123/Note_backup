### Nearest Neighbor Classifier
- 训练函数仅需记录所有学习过的数据和标签
- 预测函数寻找最相似的图像并且返回其标签

##### Distance Metric: 定义两张图片的差值，并且返回一个数。
	定义了两张图片之间的相似性，有许多不同方法可以做到这一点。
- L1 distance：求出所有像素矩阵的差，取绝对值相加作为差值。
	Q: 在N个数据的情况下，这样的算法时间复杂度是？
	A: Train O(1), but predict O(N)
	我们想要的是预测时间更少，而训练时间无所谓。

# Linear Classifier 线性分类器
- 几乎是最重要的板块，对于所有深度学习。

### 概念
- 参数方法：寻找一些参数来构建函数，产生一个映射从输入到输出的分类。
- f(x,W) = Wx + b（矩阵乘法式）
- 这是神经网络的小板块，神经网络由它堆叠而成。

### 具体原理
- 1. 将图像像素转化为一个向量
- 2. 通过 Wx + b 的乘法获得几个分类对应的数值
- 3. 这些数值对应“分数“，模型用这个来评判它与某个类型的相近程度。
- tips：
	- 对于每一个类，我们都可以在W中找到其对应的矩阵。也就是说，这种评判的标准实际上可以看作一个简单的线性函数：
	- $score = k_1*x_1 + k_2 * x_2 + ... + b$
	- 分数的构成来源于每一个像素和它权重的简单乘积，最后加上一个常数。
	- 从几何上来说，这类似于在二维中找到“线”，使得在一边分数小于阈值，它就不被评价为该类别。反之则被评价为该类别。(f = Wx + b)
	- （更高维中是一种二分的超平面）
	- b的作用是调整这个“线”的高度，否则它必定过原点的话就非常受限制了。

### 挑战：
- 它不能分类很多简单的图像。（比如不能分类1、3象限等等）
- 本质上它简单的算法只是简单的将空间二分，从而让很多复杂的图像无法分类。

### 如何获得好的W
- 定义一个损失函数(Loss func)：它直接定义了一个模型的好坏（一种“残差”之类的东西）。
- 找到一种方法，有效地改变W使得找到最小化的损失函数情况下的W。

### Loss func 的定义：
- Softmax classfier：将“评分”视为“概率”。用指数和归一化的方法得到一系列概率，总加和为1。
	- 它不仅归一化了参数，而且还用指数的形式扩大了这个差距，让更大的一项更高概率。
	- 此时，我们可以使用 L = -log(p) 的方式对正确的概率评分。
		- L = - log P，这是一个最大似然估计
		- 对比正确的概率，也就是1.00。损失函数在这个意义上和测试、正确结果两者的交叉熵一致，所以在深度学习中这个也被叫做二元交叉熵。

Q：Li的可能最大值和最小值？
A：最大正无穷，最小0
Q：当我们随机从一个W开始时，所有分数都是相等的，那么此时的L是多少？
A：-log(1/c)

### Softmax function
- soft max 顾名思义，可以分为两个部分：soft和max
	- max的含义：该函数将输出“最大值”
	- soft的含义：相比于普通的hardmax，即max()，仅仅输出数组最大值。softmax通过指数变换和归一化，将输出变成一个类似概率的形式。
- softmax函数：$$Softmax(z_i)=\frac{e^{z_i}}{\sum_{j=1}^n e^{z_j}}$$
	- 解析：先对目标分数(score)求其指数函数，再除以所有求得的指数分数总和进行归一化，得到一个新的分数。此时，所有分类的分数总和为1，且由于指数函数的引入，线性的关系被拉得更大。（大的更接近1，小的更接近0）
- 引入指数的优点：指数形式求导很方便。缺点：数值会溢出（解决方法：每个值都减去最大值即可）
```python
import numpy as np

scores = np.array([123, 456, 789])
scores -= np.max(scores)
p = np.exp(scores) / np.sum(np.exp(scores))
print(p) # [5.75274406e-290 2.39848787e-145 1.00000000e+000]
```

### Loss function
- 定义一个lossfunc的方式
	- 1.分类误差率
		- 公式：$classification\ error = \frac{count\ of\ error\ items}{count\ of\ all\ items}$
		- 问题：两个模型如果分数差距一个大一个小，但是结果是一样的，它们的loss也是一样的，就区分不出来了。
	- 2.均方误差
		- 公式：$MSE = \frac{1}{n} \sum_{i=1}^{n}(\hat{y_i}-y_i)$
		- 问题：在分类问题中，使用sigmoid/softmax得到概率，配合MSE损失函数时，采用梯度下降法进行学习时，会出现模型一开始训练时，学习速率非常慢的情况

- Cross-Entropy Loss 交叉熵
	- 表达式：
		- 二分类：$$L=\frac{1}{N}\sum_iL_i=-\frac{1}{N}\sum_i[y_i·log(p_i)+(1-y_i)·log(1-p_i)]$$
		- 多分类：$$L=\frac{1}{N}\sum_iL_i=-\frac{1}{N}\sum_i\sum_{c=1}^My_{ic}·log(p_{ic})$$
